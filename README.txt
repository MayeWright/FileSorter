# FileSorter
A file sorting python program
This program will look for a text file called Sort me.txt then sort and output the sorted contents
When this program runs it shall pull the Sort Me.txt file out of ToSort folder then ask the user for the mode to run the sorter in forward, reverse, or test
The program in forward sort mode will sort the contexts of Sort Me.txt in alphabetically order then put the sorted contents into the output.txt in the output folder
The program in backward sort mode will sort the contexts of Sort Me.txt in reverse alphabetically order then put the sorted contents into the output.txt in the output folder
The program in the test mode will run both sorts on the TestTextUnsorted.txt in test folder then conpares the results to the Forward and Backward test text files and print the results on the terminal
